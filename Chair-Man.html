Chair-Man:
A Unity game made for Android

Overview

In my spare time I have been working with the Unity game engine to build a simple 2-D game. The game idea is simple, the player is dropped into a level and they have to make their way through an obsticle course of various enemies before finding the gold coin. Once the player finds the coin they complete the level. Before starting this game my Unity knowledge was limited, I had previously completed some small projects with extremely simple ideas but this project is my first fully fleshed out game. The game featured 20 levels.

Tools Used

As mentioned the Unity game engine was used to develop this game. This engine is fantastic as it has a huge amount of learning resources due to its popularity. There are many resources found throughout the web and there are some great youtube tutorials to help get you started. Follow this link to get started with Unity.

The programming was done using C#, which is a fantastic language to use. If you do not know C# but know another programming langauge, you can easily pick it up. If you don't know any programming langauge don't worry either, because there are plenty of online resources to get started to help you on your journey of developing your own game.

I am not an artist but I was able to create artwork that I was happy with by using Krita. Krita is an open source painting program that is easy to use and has plenty of tutorials that anyone can follow. It is not hard to quickly become skilled at using Krita and use it to create all kinds of artwork. Follow this link to download Krita.

Menu Layout



World Menu

This menu displays the different worlds, which are a group of ten levels that have a similiar look to them. Initially the blue world is the only available option but once you complete each world the next world will be unlocked.

Level Menu

The level menu displays the group of ten levels that are found in each world. Each level is displayed as a button and the colour of the button shows weather the player has completed the level, unlocked the level or the level is locked. As the user completes each level more levels become unlocked.

Gameplay



Standard Levels

For most levels the objective is to avoid the various obstacles (i.e lasers and enemies) and find their way to the gold coin. The player will need to find keys or press buttons to open some of the doors found within the level.

Alternate Levels

The alternate levels involve the player moving at high speed through the level and avoiding various obstacles in their path. Once the player crosses the finish line the level is complete.
